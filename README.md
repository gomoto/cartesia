# Cartesia

- Add objects (currently lines and spheres)
- Remove objects
- Select objects
- Scale objects
- Translate objects
- Color objects
- Toggle gridlines
- Adjust gridlines
- Color gridlines
- Color background
- Undo/Redo

## Shortcuts

| Command | Keys |
| ------- | -------- |
| Undo | `Cmd Z` |
| Redo | `Shift Cmd Z` |
| Move +x | `Alt →` |
| Move -x | `Alt ←` |
| Move +y | `Alt ↑` |
| Move -y | `Alt ↓` |
| Move +z | `Alt +` |
| Move -z | `Alt -` |
| Remove object | `Del` |
